# Speech Recognition Assistant

A simple Python project using speech recognition and text-to-speech synthesis to create a voice-controlled assistant.

## Features

- **Voice Commands:** Control your computer using voice commands.
- **Open Applications:** Open specified applications by speaking their names.
- **Open Websites:** Open specified websites by mentioning their names.
- **Time Inquiry:** Ask the assistant for the current time.

## Requirements

- Python 3.x
- SpeechRecognition library (`pip install SpeechRecognition`)
- PyAudio library (`pip install pyaudio`)
- win32com.client library (`pip install pywin32`)
